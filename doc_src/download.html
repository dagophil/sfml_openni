Hau den Maulwurf für Windows

Hinweis: In dieser Version wird das Hau den Maulwurf Spiel mit der Maus bedient, nicht mit der Kinect.

(1) Zum Ausführen wird die Visual C++ Redistributable für Visual Studio 2015 (vc_redist.x86.exe) benötigt.

(2) Laden Sie Hau den Maulwurf für Windows herunter. Anschließend können Sie das Archiv entpacken und das Spiel über hdm.exe starten.
