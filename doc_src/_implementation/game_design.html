Spiel Design

Das Spiel wird mithilfe einer Endlosschleife ausgeführt. In dieser wird zuerst überprüft, ob das Programm oder Fenster geschlossen und dadurch beendet wurden.
Falls nicht, so empfängt das Spiel die aktuellen Daten der Kinect und speichert die beiden Handpositionen und überprüft ob ein Objekt gehovert wird, um ein entpsrechendes Event auslösen zu können.
