Libraries!

In unserem Projekt werden drei Libraries eingesetzt. Die Schnittstelle zur Kinect ist in der OpenNI Library implementiert. Zum Erzeugen eines Fensters und zum Laden und Zeichnen von Bildern wird SFML eingesetzt. Das Laden von XML-Dateien wird von TinyXML-2 übernommen.

OpenNI

OpenNI wurde entwickelt, um die Entwicklung von Anwendungen mit 3D Kamera Input zu erleichtern. Durch die OpenNI API können Anwendungen, zum Beispiel Spiele, unabhängig von dem Gerät entwickelt werden, das die Eingabedaten liefert. Das Hauptziel von OpenNI ist, ein Interface für Anwendungen zur Verfügung zu stellen, die ein Natural User Interface zur Eingabe nutzen (Gesten, Körperhaltungen, usw).

Die OpenNI API stellt Generatoren zur Verfügung, mit denen die konkreten Eingabedaten ausgelesen werden können. Beispielsweise wird der Zugriff auf das Tiefenbild der Kinect Kamera durch die Klasse DepthGenerator ermöglicht. Wie wir OpenNI in unserem Projekt eingesetzt haben, finden Sie unter Kinect Schnittstelle.
