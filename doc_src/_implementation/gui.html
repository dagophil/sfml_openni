Grafische Benutzeroberfläche

Wie in Libraries beschrieben, wird SFML zum Zeichnen der Benutzeroberfläche benutzt. Diese Library kann plattformunabhängig ein Fenster erzeugen und dort primitive Formen (Kreise, Rechtecke, ...) sowie Bilder einzeichnen.
